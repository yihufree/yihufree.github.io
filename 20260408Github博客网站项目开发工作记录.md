### 技术实现

使用了以下 Jekyll 模板语法：

```liquid
{% assign postsByYear = site.posts | group_by_exp:"post", "post.date | date: '%Y'" %}
{% for year in postsByYear %}
[{{ year.name }} 年](#{{ year.name }}-年) {% endfor %}

{% assign sorted_tags = site.tags | sort: 'last' | reverse | slice: 0, 10 %}
{% for tag in sorted_tags %}
[{{ tag[0] }}](#{{ tag[0] }}) {% endfor %}

| 序号 | 标题 | 日期 | 标签 |
|------|------|------|------|
{% for post in site.posts limit:10 %}
| {{ forloop.index }} | [{{ post.title }}]({{ post.url }}) | {{ post.date | date: "%Y-%m-%d" }} | {% for tag in post.tags %}`{{ tag }}` {% endfor %} |
{% endfor %}
```
