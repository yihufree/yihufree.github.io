# 笔记文档模板

## 什么是笔记文档模板？

这是一个用于创建学习笔记的标准模板，帮助你保持一致的写作风格和结构。

## 模板结构

### 1. 前置信息（Front Matter）

```yaml
---
layout: post
title: 文章标题
date: YYYY-MM-DD
tags: [标签1, 标签2, 标签3]
---
```

### 2. 文章主体

- **主标题**：使用 `#` 标记
- **章节标题**：使用 `##` 标记
- **小节标题**：使用 `###` 标记
- **正文内容**：使用普通文本

### 3. 格式建议

- 使用 Markdown 语法
- 适当使用列表、表格、代码块
- 保持段落清晰，避免过长

## 使用方法

1. 复制此文件
2. 修改文件名（遵循 YYYY-MM-DD-标题.md 格式）
3. 修改 front matter 中的标题、日期和标签
4. 替换模板内容为你的实际笔记

## 示例内容

### 代码示例

```python
# Python 代码示例
def hello():
    print("Hello, World!")

hello()
```

### 列表示例

- 学习目标
- 核心概念
- 实践应用
- 总结反思

## 总结

使用统一的模板可以帮助你：

1. 保持笔记的一致性
2. 提高写作效率
3. 便于后续查找和复习

希望这个模板对你的学习有所帮助！
